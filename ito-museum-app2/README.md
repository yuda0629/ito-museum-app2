# 伊都国遺跡デジタルマップ

Python(FW：streamlit)とR言語(FW：Shiny)で作成した遺跡可視化アプリです。

## 特徴
- 遺跡の位置表示
- 時代・種類フィルタ
- 緯度・経度の表示

## 技術
- R / Shiny / leaflet（OSSの地図ライブラリ）
- Python / streamlit
